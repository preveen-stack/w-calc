  • Wcalc 1.5 (MacOS X)(Linux-x86)(Linux-ppc)
    • Added conversions for *lots* of stuff (GUI)
    • Added support for comments (#, //, /* */) (BOTH)
    • Prints prefixes (0x, etc) by default. (CLI)
    • Added left and right bitshifts (<< and >>) (BOTH)
    • '=' and 'clear' buttons work like Calculator.app (GUI)
    • Separator preferences (commas vs decimals, etc) fetched from System Preferences (GUI)
    • Specify separators on the fly (CLI)
    • Added a preference to ignore erroneous expressions (GUI)
    • Saves drawer states (GUI)
  • Wcalc 1.4.5 (MacOS X)(Linux-x86)(Linux-ppc)
    • Rewrote the number parser. Now handles commas correctly in all cases (i.e. 1,000 is either 1000 or 1 depending on the comma preference). Flags confusing numbers (e.g. 098 or 1.0.1). (BOTH)
    • Command-line commands (like \commas or \hex) now work everywhere - the GUI as well as in files that you pipe into the CLI. (BOTH)
    • Added support for some more symbols. (BOTH)
    • Rounding indicator in the GUI (GUI)
    • Added a base drawer (GUI)
    • Preference panel and keyboard panel placement make sense now (GUI)
    • Fixed two crashes in the output routines. (BOTH)
    • CLI prints all errors now (CLI)
    • Fixed menu's tracking of the state of the drawer (GUI)
